Hello NDK
====
NDKを使ったサンプルです．  
フィボナッチ数列の第n項を求めるために必要な時間が，ネイティブコードによってどれだけ短縮できるか感じられます．  
実機（103SH）で試したところ，体感で5~10倍速くらいでてるかなあと．

--------

自分の持ってるやつ（Softbank103SH）以外で試してないのでどうなるかはわかりません．
